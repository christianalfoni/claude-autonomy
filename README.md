# Make Claude autonomous on your repo

Open, self-contained setup prompts for [Claude Code](https://claude.com/claude-code) that make it work without you in the loop. Paste one into a session opened in your repo and it builds the capability into files **you own and edit** — no plugin, no install, no runtime dependency on this repo.

**→ [christianalfoni.github.io/claude-autonomy](https://christianalfoni.github.io/claude-autonomy) — browse and copy a prompt.**
