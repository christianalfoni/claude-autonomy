Claude Code plugin

Your agent should get better every session.

close-gaps reviews each working session when it ends and captures what would make the next run faster — as skills committed to your project.

Install

Adds a skill and a Stop hook — no settings.json edits.

/plugin marketplace add christianalfoni/claude-plugins
/plugin install close-gaps@christianalfoni

# then /clear, and work as usual

How it works

  1. You finish a piece of work and the session goes to stop.
  2. A Stop hook fires — once per working session, skipping trivial Q&A.
  3. It runs the close-gaps review over the session.
  4. Durable findings are committed as project skills. Next run starts ahead.

What it captures

Three kinds of skill, written to your project's own .claude/skills/ — never globally.

01 · docs

Knowledge

How to use the project's libraries and conventions, indexed as a table of contents. Anchored by a self-healing project-overview.

02 · validate

Validation tools

A generic CLI that drives the real interface — UI, command, endpoint, API — so a result is proven the way its user sees it. Never a proxy.

03 · reach

External context

A generic CLI over an external system — a DB, an issue tracker, an API. The one gap that may need you: authentication.
